🏏 IPL Player Data Analysis using Python

📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on an IPL players dataset to understand player distribution, team composition, and spending patterns.
Using Python libraries, the project cleans the dataset, analyzes player information, and visualizes key insights about teams and players.

The goal of this project is to demonstrate data analysis, data cleaning, and visualization skills using Python.

📊 Dataset

The dataset contains information about IPL players including:

Player Name

Player Type (Batsman, Bowler, All-rounder, Wicketkeeper)

Base Price

Final Auction Cost

Team

2022 Squad


⚙️ Technologies Used

Python

Pandas – Data manipulation and analysis

NumPy – Numerical operations

Seaborn – Data visualization

Matplotlib – Plotting graphs

🔍 Key Steps Performed
1️⃣ Data Cleaning

Removed unnecessary columns

Renamed columns for better readability

Handled missing values

Checked for duplicates

2️⃣ Exploratory Data Analysis

Identified most expensive players

Found cheapest players

Counted players by type

Analyzed players per team

Calculated total number of unique players

3️⃣ Data Visualization

The following visualizations were created:

Team-wise player count

Player type distribution

Team-wise spending

Top 5 most expensive players

Base price distribution

📈 Sample Insights

Some teams spent significantly more on players than others.

Bowlers and all-rounders form a large portion of team composition.

A few players dominate the highest auction price category.
